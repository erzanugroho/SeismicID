performa model

evaluasi kualitas prediksi: reliability diagram, roc, skill score vs baseline poisson, csep-style tests, dan leave-one-region-out cv.

apa arti halaman ini?

halaman ini mengukur seberapa bisa dipercaya prediksi model dibanding tebakan acak dan baseline statistik sederhana (poisson). aturan baca singkat: hijau = lebih baik dari baseline, merah = lebih buruk. tiap grafik punya penjelasan ringkas di bawahnya.

metadata model aktif

Belum ada model terlatih. Sistem berjalan dalam mode demo seed: probabilitas dihitung dari fitur fisis (jarak patahan, kedalaman slab, tipe patahan) sebagai placeholder, bukan output ML nyata. Metrik evaluasi di bawah hanya akan tampil setelah model dilatih dengan data historis (lihat scripts/bootstrap_data.py + scripts/train_initial.py).
